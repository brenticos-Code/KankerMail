# KankerMail
Standard email client made for windows or to use in a web environment using html. This is a basic project and is gonna get better day after day.


This program is made mainly using GO. The TUI program is made to run on mainly on Linux.
Since this is locally hosted software this does need to get some info from your e-mail provider.
This includes IMAP Settings
and SMTP settings.
These need an "App-Password"

Let me know if theres a problem.


(an example of running the web based ui)
"./kankermail -web -port 8080"
